# Pizza-Tower-Eggplant-On-Raspberry
What is this project?


this allows You to play the Eggplant Build

on Raspberry Pi Hardware, this version

comes with a Debug Console, Unlike The Original
Eggplant Build.

# How to set it up
go to releases,

download the ZIP file, extract it,

get your Eggplant Build files,

put them in the folder called Game/ then,

run setup.sh, and it should be in your

Home folder.
